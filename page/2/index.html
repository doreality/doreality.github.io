Python 根据源码生成类图

2021-10-13 Tools

Pyreverse

  1. 首先安装graphviz，用于绘图

  2. pyreverse能方便的生成uml类图，pylint里自带了pyreverse这个工具

    pip install pylint

  3. 使用

    pyreverse -ASmy -o esp -p _后缀 目录/

参考：python代码自动生成类图_python逆向工程：通过代码生成类图

PyCharm

在 PyCharm 中，右键点击想分析的代码，选择Diagrams->Show Diagrams，就可以生成当前代码的类图。
