  2. RESTful API

    REST = Representational State Transfer

    使用 REST 实现 Http 的 CRUD（Create、Update、Read、Delete）。

    例如：http://vidly.com/api/customers

    把 customers 看作是一种资源，对于该资源的增删改查都是通过向这个 end point 发送请求完成的。

    请求的种类（HTTP METHODS）对应着操作的类型：

    • GET
    • POST
    • PUT
    • DELETE

    使用有语义的地址来公开资源，可以使用 HTTP 的规则来对资源操作。
